RegEx

Regular expression engine.

Support for following matches:

  • . ‐ any
  • ? ‐ optional
  • * ‐ zero or more repetitions
  • + ‐ one or more repetitions
  • ^ ‐ first single match
  • $ ‐ last single match
  • group ‐ (one|two|three)
  • range ‐ [start-end]

Support for following special patterns:

  • word ‐ \w
  • not word ‐ \W
  • digit ‐ \d
  • not digit ‐ \D
  • white space ‐ \s
  • not white space ‐ \s
  • repeat ‐ {least, most}
