IIS Setup
===
* Go to "Turn Windows features on or off"
* Check "Internet Information Services"
* Next, go to "Internet Information Services" \ "World Wide Web Services" \ "Application Development Features"
	- Check "ISAPI Extensions"
	- Check "ISAPI Filters"
* Click 'Ok'
---
* Open "Internet Information Services Manager (IIS)"
	- Chick "Sites" \ "Default Web Site"
	- Right click and choose "Edit Binding", Port
	- Change the port value to "8080" 
	- Close dialog
---
Copy "objeck_iis.dll" to "%windir%\inetpub\wwwroot"

Install Module
===
* Select the top level of the Connection tree control
	- Select "Modules", "Configure Native Models" and "Register..."
	- Use name "objeck_iis" and select the path to "objeck_iis.dll"
	- Click, Ok and Ok
* Select the top level of the Connection tree control
	- Select "ISAPI and CGI Restrictions" and "Add"
	- Select the path to "objeck_iis.dll"
	- Use description name "objeck_iis" 
	- Check "Allow extension path to execute"
* Select the top level of the Connection tree control
	- Select "Handle Mapping" and "Add Module Mapping..."
	- Set "Request path" to '*'
	- Choose the "objeck_iis" module
	- Choose the "objeck_iis.dll" executable
	- Use name "objeck_iis"
	- Click on "Request Restrictions" and uncheck "Invoke handler only if request is mapped to"
	- Select Access and choose "Execute"
	- Click, Ok and Ok
* Select the top level of the Connection tree control
	- On the upper right check "Restart"
