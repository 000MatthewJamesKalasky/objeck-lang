Objeck release process
---
1. Trial build on Windows; push code
2. Stage update documentation
	a. GitHub 
		i.	LSP
		ii.	Objeck
	b. Sourceforge
1. Update system libraries on Linux x64; push code
2. Full build on Windows with LSP binaries; push code; copy binaries to network directory
3. Build other OSs and copy binaries to network directory
	a. Linux x64
	b. Linux arm64
	c. macOS x64
	d. macOS arm64
4. Run binary renaming tool
5. Upload binaries to Sourceforge and GitHub
6. Upload API documentation to objeck.org
7. Post update documentation
	a. GitHub 
		i.	LSP
		ii.	Objeck
	b. Sourceforge
	c. freshcode
