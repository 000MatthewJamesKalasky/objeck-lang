Text Editor Support

 

The following outlines how to enable code syntax highlighting for various editors. Preferred editor is Sublime, which is cross platform, and provides both syntax highlighting and build support.

 

Sublime (Windows/Linux/Mac)

Support for syntax highlighting and compiling code.

Windows: Copy the objeck.sublime-syntax, objeck.tmPreferences and objeck.sublime-build files to “%USERPROFILE%\AppData\Roaming\Sublime Text 3\Packages” directory

Restart the application

 

Linux: Copy the objeck.sublime-syntax, objeck.tmPreferences and objeck.sublime-build files to the “~/.config/sublime-text-3/Packages” directory

Restart the application

 

Mac: Copy the objeck.sublime-syntax, objeck.tmPreferences and objeck.sublime-build files to the “~/Library/Application Support/Sublime Text 3/Packages” directory

Restart the application.

 

Ensure the “build” command path is absolute or that the Objeck bin directory is in your environment path.

 

For Windows:

{

     "env": {

           "OBJECK_LIB_PATH": "<install_path>/lib",

     },

     "cmd": ["<install_path>/bin/obc", "-src", "$file", "-lib", "collect.obl,gen_collect.obl,net.obl,rss.obl,misc.obl,regex.obl,csv.obl,xml.obl,json.obl,encrypt.obl,odbc.obl,query.obl,sdl2.obl,sdl_game.obl", "-dest", "$file_base_name.obe"],

     "selector": "source.objeck-obs",

     "file_regex": "^(..[^:]*):([0-9]+):?([0-9]+)?:? (.*)$"

}

 

For Linux and Mac:

{

"env": {

     "OBJECK_LIB_PATH": "<install_path>/lib",

     "PATH": "<install_path>/bin"

},

     "cmd": ["obc", "-src", "$file", "-lib", "collect.obl,gen_collect.obl,net.obl,rss.obl,misc.obl,regex.obl,csv.obl,xml.obl,json.obl,encrypt.obl,odbc.obl,query.obl,sdl2.obl,sdl_game.obl", "-dest", "$file_base_name.obe"],

     "selector": "source.objeck-obs",

     "file_regex": "^(..[^:]*):([0-9]+):?([0-9]+)?:? (.*)$"

}

 

Notepad++ (Windows)

Support for syntax highlighting

Open application click “Language” ”Define your language”

Select “Import” and navigate to the objeck.xml file

Restart the application

 

BBEdit and Text Wrangler (Mac)

Support for syntax highlighting

Copy objeck.plist to the “/Library/Application Support/BBEdit/Language Modules/” or “/Library/Application Support/TextWrangler/Language Modules/” directory

Restart the application

Note: you may need to create this directory

 

gEdit (Linux)

Support for syntax highlighting

Copy objeck.lang to the “/usr/share/gtksourceview-3.0/language-specs/”

Restart the application
